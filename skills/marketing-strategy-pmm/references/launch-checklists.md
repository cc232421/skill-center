# Launch Checklists

## Tier Definitions

| Tier | Scope | Prep Time | Budget | Audience |
|------|-------|-----------|--------|----------|
| 1 | New product, major feature | 6-8 weeks | $50-100k | All prospects + press |
| 2 | Significant feature, integration | 3-4 weeks | $10-25k | Customers + select prospects |
| 3 | Small improvement | 1 week | <$5k | Existing customers only |

---

## Tier 1: Major Launch Checklist

### Phase 1: Foundation (Weeks 1-2)

- [ ] Kickoff meeting with Product, Marketing, Sales, CS
- [ ] Define launch goals (pipeline $, MQLs, press coverage)
- [ ] Assign RACI matrix
- [ ] Set up launch tracking dashboard
- [ ] Confirm product ship date

### Phase 2: Preparation (Weeks 3-6)

- [ ] Finalize positioning and messaging
- [ ] Create sales deck and one-pagers
- [ ] Build competitive battlecards
- [ ] Develop demo script
- [ ] Write launch press release
- [ ] Create landing page
- [ ] Prepare email sequences
- [ ] Design ad creative
- [ ] Train sales team
- [ ] Prepare CS team for inquiries

### Phase 3: Launch (Week 7-8)

- [ ] Distribute press release
- [ ] Send email announcement to database
- [ ] Publish social media posts
- [ ] Ramp paid ads to full budget
- [ ] Launch sales outbound blitz
- [ ] Enable in-app notifications
- [ ] Monitor metrics every 2 hours

### Phase 4: Post-Launch (Weeks 9-12)

- [ ] Week 2: Verify pipeline on track to goal
- [ ] Week 4: Optimize underperforming channels
- [ ] Week 8: Calculate ROMI
- [ ] Week 12: Document learnings

---

## Tier 2: Standard Launch Checklist

### Week 1

- [ ] Define launch goals
- [ ] Finalize messaging
- [ ] Create sales one-pager
- [ ] Build email sequence
- [ ] Notify existing customers

### Week 2

- [ ] Send announcement email
- [ ] Publish blog post
- [ ] Post on social channels
- [ ] Brief sales team (15 min)

### Week 3-4

- [ ] Monitor metrics
- [ ] Answer questions from customers
- [ ] Document learnings

---

## Tier 3: Minor Launch Checklist

- [ ] Announce in product (changelog)
- [ ] Send to customer newsletter
- [ ] Quick social post (if relevant)
- [ ] Monitor feedback

---

## Launch Day Runbook

### Morning (8am - 12pm)

- [ ] Verify all assets live
- [ ] Confirm ads running
- [ ] Check monitoring dashboard
- [ ] Alert support team

### Mid-Day (12pm - 6pm)

- [ ] Monitor engagement metrics
- [ ] Respond to social mentions
- [ ] Track demo request volume

### Evening (6pm - 10pm)

- [ ] Review daily metrics
- [ ] Document learnings
- [ ] Plan next day optimizations
