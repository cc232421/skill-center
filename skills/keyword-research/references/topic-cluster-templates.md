## Architecture Patterns

### Classic Hub-and-Spoke

```
PILLAR: "Project Management Software"
│
├── Cluster 1: "Project management for small business"
├── Cluster 2: "Remote team project management"
├── Cluster 3: "Project management templates"
└── Cluster 4: "Agile project management"
```

### Layered Architecture

```
PILLAR: "Marketing Strategy"
│
├── Layer 1: Core topics (broad)
│   ├── "Digital marketing strategy"
│   └── "Content marketing strategy"
│
├── Layer 2: Specific topics
│   ├── "B2B marketing strategy"
│   ├── "SaaS marketing strategy"
│   └── "Local marketing strategy"
│
└── Layer 3: Long-tail
    ├── "Marketing strategy for startups"
    └── "Marketing strategy for restaurants"
```

### Use-Case Architecture

```
PILLAR: "CRM Software"
│
├── Use Case 1: Sales
│   ├── "Sales CRM for small business"
│   └── "CRM sales pipeline management"
│
├── Use Case 2: Marketing
│   ├── "Marketing CRM automation"
│   └── "CRM email marketing"
│
└── Use Case 3: Customer Service
    ├── "CRM customer support"
    └── "CRM helpdesk integration"
```

### Process Architecture

```
PILLAR: "How to Do Keyword Research"
│
├── Step 1: Seed keyword generation
├── Step 2: Keyword expansion
├── Step 3: Intent classification
├── Step 4: Difficulty analysis
├── Step 5: Opportunity scoring
└── Step 6: Cluster building
```

## Internal Linking

### Anchor Text Strategy

| Link Type | Anchor Text % | Example |
|-----------|---------------|---------|
| Exact match | 20-30% | "project management software" |
| Partial match | 40-50% | "project management tools" |
| Branded | 20-30% | "Acme software" |
| Generic | 10-20% | "learn more", "click here" |

### Link Density Guidelines

| Content Type | Internal Links | Outbound Links |
|-------------|---------------|----------------|
| Pillar page | 15-25 | 5-10 |
| Cluster content | 8-15 | 3-5 |
| Blog post | 5-10 | 2-5 |

## Content Calendar Integration

### Build Sequence

1. **Weeks 1-2**: Publish pillar content
2. **Weeks 3-6**: Publish cluster content (2-3 per week)
3. **Weeks 7-8**: Audit links, fix gaps, optimize

### Publishing Cadence

| Site Authority | Cluster Size | Publish Frequency |
|---------------|--------------|-------------------|
| New site | 5-10 clusters | 1 pillar + 2-3 clusters/month |
| Medium | 10-20 clusters | 2 pillars + 4-6 clusters/month |
| Established | 20+ clusters | 3 pillars + 8-10 clusters/month |

## Performance Measurement

### Cluster Health Score

| Metric | Target | Weight |
|--------|--------|--------|
| Cluster keywords ranking | Top 20 for 60%+ | 30% |
| Cluster internal links | All clusters linked | 25% |
| Content freshness | Updated within 6 months | 20% |
| Traffic to cluster | Growing MoM | 15% |
| Conversion from cluster | >1% | 10% |

### Review Cadence

- **Weekly**: Check rankings, traffic anomalies
- **Monthly**: Review cluster performance, plan additions
- **Quarterly**: Full cluster audit, refresh strategy
