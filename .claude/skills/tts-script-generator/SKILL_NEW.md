# TTS Script Generator

将长文档智能压缩为适合 TTS 朗读的脚本。

## 快速开始

**用户说：**
```
"请用 tts-script-generator 改写这篇文章"
```

**Claude 会：**
1. 分析内容，决定最佳时长（3-8分钟）
2. 压缩并口语化改写
3. 自动分段（每段15秒左右）
4. 输出 `tts_script.json`

## 输入输出

**输入：** 长文档（txt/md/纯文本）

**输出：** `tts_script.json`
```json
[
  {
    "segment": 1,
    "text": "如果让你穿越，你会选哪个朝代？大部分人都会说：唐朝！",
    "duration": 15
  }
]
```

## 工作原理

### 1. 智能分析
- 计算原文字数和时长
- 判断最佳视频时长（3-8分钟）
- 确定压缩比例

### 2. 智能改写
- 提取核心观点
- 压缩到目标长度
- 转换为口语化表达
- 添加情感词语

### 3. 自动分段
- 每段10-20秒
- 每段一个核心点
- 适合配一张图

## 改写原则

**口语化：**
- "但是" → "但是呢"
- "因为...所以..." → "为什么呢？因为..."
- 添加语气词："啊"、"呢"、"吧"

**情感优化：**
- 强调词："真的"、"非常"、"特别"
- 惊叹词："哇"、"天哪"、"想象一下"

**节奏控制：**
- 短句为主
- 适当停顿
- 清晰表达

## 示例

**原文（5947字，19分钟）：**
```
今天我们来讲中国古人的真实生活。在讲之前我有个问题...
```

**输出（1350字，5分钟，20段）：**
```json
[
  {"segment": 1, "text": "如果让你穿越，你会选哪个朝代？...", "duration": 15},
  {"segment": 2, "text": "但是，你知道吗？唐朝没有卫生纸！...", "duration": 15}
]
```
